# BlinkIT-Grocery-Data-Dashbord
Power BI Dashboard Project | BlinkIT Grocery Sales Analysis
where I analyzed grocery sales data from BlinkIT to derive meaningful business insights and present them through an interactive and visually appealing dashboard. This project is now uploaded to GitHub for reference and learning.
**Project Objective:**
 The main goal of this project was to understand sales performance, customer behavior, and product trends across different outlets and categories, enabling data-driven decision making.
**Key KPIs & Insights Covered:**
 • Total Sales – Overall revenue generated
 • Average Sales – Average sales per transaction
 • Average Rating – Customer satisfaction indicator
 • Number of Items – Total items sold
 • Category-wise sales analysis
 • Outlet-wise performance comparison
 • Interactive filters for better exploration and insights
**Dashboard Features:**
 Dynamic visuals with slicers for easy data exploration
 Clean and consistent dashboard layout
 KPI cards for quick decision-making
 Beginner-friendly yet business-focused analysis
**Tools & Skills Used:**
 • Power BI
 • Data Cleaning & Transformation
 • Data Modeling
 • DAX (basic calculations & measures)
 • Data Visualization & Storytelling
**What I Learned From This Project:**
 • How to convert raw data into meaningful insights
 • Designing professional dashboards with business KPIs
 • Improving visualization clarity and user experience
 • Strengthening practical Power BI and analytical skills
